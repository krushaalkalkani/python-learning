# CRM System

A full-stack Customer Relationship Management application with React frontend and Node.js/Express backend.

## Features

- **Dashboard** — Overview stats, recent orders, revenue
- **Products** — CRUD, stock tracking, categories
- **Customers** — Contact management
- **Orders** — Create orders, status updates, inventory deduction
- **Analytics** — Charts, top products, revenue trends

## Quick Start

### 1. Backend

```bash
cd CRM/backend
npm install
npm start
```

Runs on **http://localhost:5000**

### 2. Frontend (new terminal)

```bash
cd CRM/frontend
npm install
npm start
```

Opens **http://localhost:3000**

## Move to Desktop (optional)

Copy the `CRM` folder to your Desktop:

```bash
cp -R CRM ~/Desktop/CRM
```

## Tech Stack

- Frontend: React 18, React Router, Axios, Recharts
- Backend: Node.js, Express, SQLite
